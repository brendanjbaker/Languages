# Program

* Program name: `sum`

## Description

This program should print the sum of 0-100 (inclusive), which is `5050`, with a trailing newline, and exit with status `0`.

There are several straightforward ways to do this programmatically:

* Looping/counting:
  * `for n in 0 to 100 ...`
* List/set operations:
  * `Range(0, 100).Sum()`
* Algebraically:
  * `100 * 101 / 2`
* Manually:
  * `0 + 1 + 2 + 3 + ... + 100`

To demonstrate basic program logic/control, I've opted to implement most programs using a looping approach.

For some languages, I've included more than one approach (but of course, only one is called to generate output).
